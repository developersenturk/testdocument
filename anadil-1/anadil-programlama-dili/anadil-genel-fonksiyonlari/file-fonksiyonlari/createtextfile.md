# CreateTextFile

Bu fonksiyon Text dosya yaratır.\
\
Kullanımı

```
Function CreateTextFile(
  FileName As String  'Filename
)As Object
```

Parametreler\
_FileName_\
Yaratılacak dosyanın adını içeren String ifadedir.\
\
Geri Dönüş Değeri\
Geri dönüş değeri Object olup, yaratılan dosyanın object ifadesidir.\
