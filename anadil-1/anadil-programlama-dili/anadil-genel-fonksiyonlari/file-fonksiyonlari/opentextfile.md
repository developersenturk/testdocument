# OpenTextFile

Bu fonksiyon Text dosyayı açar.\
\
Kullanımı

```
Function OpenTextFile (
  FileName As String  'Filename
)As Object
```

Parametreler\
_FileName_\
Açılacak dosyanın adını içeren String ifadedir.\
\
Geri Dönüş Değeri\
Geri dönüş değeri Object olup, açılan dosyanın object ifadesidir.\
