# LockSecureObject

Bu fonksiyon string olarak verilen object i lock eder.\
\
**Kullanımı**

```
Function LockSecureObject(
ObjectString As String
) As Bool
```

\
**Parametreler**\
_ObjectString_\
Lock edilecek object string parametresi.\
\
**Geri Dönüş Değeri**\
Object lock edildiğinde TRUE, edilemezse FALSE değeri döner.\
