# Sleep

Bu fonksiyon verilen milisaniye kadar bekleme yapar.\
\
Kullanımı

```
Function Sleep(
  Milliseconds As Number   '  1000 : 1 saniye   10000 : 10 saniye 
)As Bool
```

Parametreler\
_Milliseconds_\
Bekleme süresidir.\
\
Örnek

```
Sleep( 1000 ) ' 1 saniye
Sleep( 60000 ) ' 1 dakika
Sleep( 3600000 ) ' 1 saat
```
