# CloseDevice

Bu method açılmış cihaz nesnesini kapatır..\
\
**Kullanımı**

```
DeviceObject.CloseDevice(
) As Bool
```

**Parametreler**\
Parametresi bulunmamaktadır.\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri, işlem başarılı olduğunda TRUE, değilse FALSE ifadedir.\
\
