# ReportJob

ReportJob Methodu, bir raporu print ettirme işlemini yapar.\
\
**Kullanımı**

```
AlwaysCallObject.ReportJob(
  ReportName As String  ' Rapor adı
) As Bool
```

**Parametreler**\
_ReportName_\
Print ettirilecek rapor adı\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri booleandır.\
\
**Dikkat Edilecek Noktalar**\
ReportJob Methodu, bir raporu print ettirme işlemini yapar.
