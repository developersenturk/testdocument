# ReportPreview

ReportPreview Methodu, bir raporu öngörmeye alır.\
\
**Kullanımı**

```
AlwaysCallObject.ReportPreview(
  ReportName As String  ' Rapor adı
) As Bool
```

**Parametreler**\
_ReportName_\
Çalıştırılmak istenen rapor adı\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri booleandır.\
\
**Dikkat Edilecek Noktalar**\
ReportPreview Methodu, bir raporu öngörmeye alır.
