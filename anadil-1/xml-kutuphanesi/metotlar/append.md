# Append

Bu method XML nesnesine parametre olarak verilen string ifadeyi eklemektedir.\
\
**Kullanımı**

```
XMLObject.Append(
   szXMLString As String
   ) As Bool
```

**Parametreler**\
\
_szXMLString_\
XML nesnesine eklenecek string ifade\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri işlem başarılı ise TRUE, değilse FALSE ifadedir.
