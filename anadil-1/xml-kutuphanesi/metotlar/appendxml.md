# AppendXML

Bu method XML nesnesine, bir başka XML nesnesini eklemektedir.\
\
**Kullanımı**

```
XMLObject.AppendXML(
   oXML As Object
   ) As Bool
```

**Parametreler**\
\
_oXML_\
XML nesnesine eklenecek XML nesnesi\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri işlem başarılı ise TRUE, değilse FALSE ifadedir.\
