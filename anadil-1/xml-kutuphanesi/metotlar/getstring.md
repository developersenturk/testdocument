# GetString

Bu method XML nesnesinden, string ifadenin alinabilmesini sağlar.\
\
**Kullanımı**

```
XMLObject.GetString(
   nStart  As Number
   nLen    As Number
   ) As String
```

**Parametreler**\
\
_nStart_\
Başlangıç pozisyonu\
\
_nLen_\
Başlangıç pozisyonundan itibaren alınacak string genişliği\
\
**Geri Dönüş Değeri**\
Geri dönüş değeri nLen genişliğinde string ifadedir.\
